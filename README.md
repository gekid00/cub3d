cub3D

Petit projet cub3D - rendu simple avec MiniLibX.

Aperçu
- Projet pour afficher une scène 3D basique avec MiniLibX.

Dépendances
- MiniLibX (fournie ou installée sur le système)
- make, gcc

Compilation
1. Se placer à la racine du projet :

    cd cub3d
    make

2. L'exécutable sera créé : ./cub3D

Exécution
- Exemple :

    ./cub3D test_maps/map.cub

Remarques
- Si la compilation échoue sur mlx.h, assurez-vous que MiniLibX est présente dans le dossier cub3d ou installée globalement.
